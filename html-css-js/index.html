Live & Dead Forensics Project

This project was part of the EECS 388 course, Introduction to Computer Security. In this project, we were given a virtual machine with a linux operating system that had been recovered. We were tasked with finding the incriminating evidence on the machine, and then writing a report on what we found. This project heavily utilized python to conduct various attacks on (instructor created) webstites as more information behind the perpetrator was discovered, including but not limited to: SQL injection, cross site scripting, and password cracking.
